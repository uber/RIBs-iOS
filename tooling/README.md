# CombineRIBs Xcode Templates

We have created Xcode templates to generate CombineRIBs scaffolding and test scaffolding, making CombineRIBs usage and adoption easier. The scaffolded classes have CombineRIBs wired up, ready to add business logic to them.

For details on installation and usage, please see the original [iOS tooling wiki page](https://github.com/uber/RIBs/wiki/iOS-Tooling) for reference.
